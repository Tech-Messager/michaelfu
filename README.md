# Portfolio Website

## Deployment Instructions

1. **Clone the Repository**  
   Use the command:
   ```bash
   git clone https://github.com/Tech-Messager/michaelfu.git
   ```

2. **Install Dependencies**  
   Navigate to the project directory and run:
   ```bash
   npm install
   ```

3. **Build the Project**  
   Run the following command to compile the project:
   ```bash
   npm run build
   ```

4. **Start the Server**  
   Launch the server with:
   ```bash
   npm start
   ```

5. **Access the Website**  
   Open your browser and go to `http://localhost:3000` to view your portfolio website.

## Customization Guide

- **Update Content**:  
  Modify the `src/content.js` file to customize texts and descriptions shown on your portfolio.

- **Change Theme**:  
  Navigate to the `src/styles` directory and update the CSS files to change styles and themes. Explore different color schemes and layouts as per your preference.

- **Add Projects**:  
  To add new projects, update the `projects` array in the `src/content.js` file with details about your work.

- **Host Online**:  
  Consider using services like GitHub Pages, Vercel, or Netlify for deploying your portfolio website online.
